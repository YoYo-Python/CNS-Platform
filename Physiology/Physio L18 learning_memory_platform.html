Learning & Memory

Master synaptic mechanisms, memory classification, consolidation pathways, and clinical disorders. Covers non-associative & associative learning, STM/ITM/LTM mechanisms, hippocampal consolidation, amnesia, and Alzheimer's disease.

⭐ KEY INSIGHT — Classical Conditioning = Intermediate Memory (Same Mechanism!)

Serotonin → ↑cAMP → K⁺ channel closure → prolonged AP → massive Ca²⁺ influx. This is the same cascade for BOTH classical conditioning AND intermediate-term memory. Exam-favorite MCQ!

Non-Associative Learning

Habituation: ↓ Ca²⁺ in nerve endings → decreased response to repeated non-significant stimulus.
Sensitization: Prolonged action potential → enhanced response when coupled with pleasant/unpleasant stimulus.

MCQ ⚠️
🔗

Associative Learning

Classical conditioning: US → excitatory interneurons → serotonin → cAMP → K⁺ closure → ↑Ca²⁺. Operant: animal operates task for reward/punishment. Avoidance: paired with negative stimulus.

Written Q ⚠️
⚡

Short-Term Memory

Duration: seconds to minutes. Mechanism: Reverberating neuronal circuits or post-tetanic potentiation. Functional, not structural. Requires consolidation to persist.

🧪

Intermediate-Term Memory

Duration: hours, days, or weeks. Mechanism: Chemical changes — serotonin → cAMP → K⁺ closure → Ca²⁺ entry. Bridge between STM and LTM.

Written Q ⚠️
🏛️

Long-Term Memory

Duration: years–lifetime. Mechanism: Structural changes — ↑vesicle release sites, ↑presynaptic terminals, ↑transmitter vesicles, altered dendritic spines.

Written Q ⚠️
🦛

Hippocampus & Consolidation

The master regulator of memory consolidation. Drives recycling/rehearsal of new information. Requires ≥1 hour of rehearsal to convert STM → LTM. Stores in cortical association areas.

MCQ ⚠️
⚠️

Amnesia Types

Retrograde: Cannot recall past LTM (good present recall). Anterograde: Cannot form new LTM (hippocampal/medial temporal lesion). Good past recall, poor recent recall.

🔴

Alzheimer's Disease

Progressive, irreversible neurodegeneration. Loss of synapses, neurons, and acetylcholine. Starts with progressive STM loss → general cognitive decline (senile dementia).

⚠️ Clinical Pearl — Alzheimer's Target

Loss of acetylcholine is the primary neurotransmitter deficit → rationale for cholinesterase inhibitors (donepezil, rivastigmine). Starts with STM loss because STM-to-LTM consolidation requires intact cholinergic synapses.
