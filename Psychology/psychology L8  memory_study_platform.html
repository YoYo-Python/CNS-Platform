⚡ Quick Summary Cards
The four foundational topics — master these first
🔄
Topic 1

Information Processing Model

Memory has 3 processes: Encoding (input), Storage/Consolidation (retain), and Retrieval (recall/recognition/relearning). All information flows: Environment → Sensory → STM → LTM.

⏱️
Topic 2

Sensory & Short-Term Memory

Sensory: <1 second, unlimited capacity. Types: iconic (visual), echoic (auditory). STM (Working Memory): 15-30 seconds, 7±2 items. Extended via rehearsal. Used for problem-solving.

♾️
Topic 3

Long-Term Memory & Brain Areas

Unlimited duration & capacity. Explicit: Semantic (facts) + Episodic (events) — hippocampus. Implicit/Procedural: Skills — cerebellum/motor cortex. Amygdala = emotional memories.

🚫
Topic 4

Forgetting & Study Economy

3 theories: Interference (new disrupts old), Repression (painful memories blocked), Disuse Atrophy (unused fades). Fix with: spaced repetition, chunking, mnemonics, recitation, sleep.

🕐 Memory Timeline
The journey of information through your brain
🌍 Environmental Input
Light, sound, touch, smell enter the senses simultaneously — all raw data.
👁️ Sensory Memory (<1 second)
Iconic (visual) or echoic (auditory) impression retained briefly. Vast majority lost immediately without attention. Gateway filter of the brain.
🎯 Attention Filter
Only attended information moves forward. This is why focusing during lectures is critical — inattention = loss at this stage.
💭 Short-Term Memory / Working Memory (15–30 sec)
Capacity: 7±2 items. Used for active problem-solving, oral exams, calculations. Fragile — distraction causes loss. Prefrontal cortex.
🔁 Rehearsal / Consolidation
Verbal repetition, recitation, spaced review. The hippocampus converts STM → LTM. Sleep enhances this process.
🏛️ Long-Term Memory (Lifelong)
Semantic (facts), Episodic (events), Procedural (skills). Unlimited capacity. Subject to disuse atrophy and interference if not reviewed.
⚠️ High-Yield Clinical Signs
Exam-favorite facts — know these cold
🔴 Exam Favourite
🏛️ Hippocampus Lesion
Prevents STM → LTM consolidation. Patient cannot form NEW declarative memories = anterograde amnesia (e.g., blackout, dementia pattern).
⭐ High-Yield
7 ± 2 Rule
STM holds exactly 7 ± 2 items (5-9). Chunking groups items to beat this limit (e.g., phone number 1485-0385 = 2 chunks, not 8 digits).
💎 Clinical Pearl
Recall vs Recognition
Recall = info NOT present (written/oral exam). Recognition = info present to senses (MCQ, practical anatomy). Recall is harder.
📚 Study Tip
Spaced > Massed Repetition
Spaced repetition requires fewer trials than cramming. Take 10-15 min breaks, switch subjects to avoid interference.
⭐ High-Yield
Iconic & Echoic Memory
Iconic = visual sensory memory. Echoic = auditory sensory memory. Both last <1 second. Part of sensory stage, not STM.
🔴 Exam Favourite
🔮 Amygdala & Emotion
Amygdala = emotional memories. Drives repression of painful experiences. Lesion = blunted emotional recall, cannot associate memories with fear/joy.
💎 Clinical Pearl
🚴 Procedural Memory
Riding a bike, driving a car = procedural (implicit). Stored in cerebellum + motor cortex. Unconscious — does NOT require hippocampus.
📚 Study Tip
💤 Sleep & Memory
Forgetting slows during sleep — no new interference. Sleep is active processing time for consolidating LTM. Never sacrifice sleep before exams.
⭐ High-Yield
💪 Disuse Atrophy
Memory traces weaken if unused — like a muscle. Retired athlete loses sport skills after years of inactivity. Regular review prevents this.

💊 Forgetting Mechanisms
Three theories of why we forget
🔀

Interference Theory

New learning disrupts and overwrites older memory traces. Studying too many subjects consecutively without breaks causes interference. Solution: subject-switching + breaks.

🔒

Repression Theory

The brain actively blocks painful or distressing memories, removing them from conscious access. Driven by the amygdala. Example: forgetting traumatic event details.

⏳

Disuse Atrophy

Memory traces biologically weaken if not used over time — just like muscle atrophy. Example: retired athlete losing sport skill after 5 years. Solution: regular review.

📚 Economy in Memorizing
Evidence-based strategies to learn more in less time
🥇 Most Effective
HIGH IMPACT

⏰ Spaced Repetition

  • Spread study sessions across multiple days
  • Requires fewer total trials than massed repetition
  • Take 10–15 minute breaks between subjects
  • Prevents interference between subjects
  • Sleep between sessions greatly enhances consolidation
⚡ STM Booster
HIGH IMPACT

🧩 Chunking

  • Group individual items into meaningful units
  • Beats the 7±2 STM capacity limit
  • Phone numbers: 1485-0385 (2 chunks, not 8 digits)
  • Patient IDs, drug doses, anatomy lists all benefit
  • Combine with mnemonics for maximum effect
🗣️ Active Recall
MEDIUM IMPACT

🔊 Recitation

  • Explain the material out loud to yourself
  • Far superior to passive re-reading
  • Forces active retrieval, strengthening memory traces
  • Simulates oral exam conditions
  • Teach-back method: explain as if to another student
🔗 Association
MEDIUM IMPACT

💡 Mnemonics

  • Connect new info to existing strong memories
  • Acronyms, rhymes, visual images, stories
  • Example: SPIKES for breaking bad news
  • Works by creating multiple retrieval pathways
  • More pathways = easier recall under exam stress
🧘 Focus
FOUNDATION

🚫 Preventing Distraction

  • Distraction during STM phase = permanent loss
  • Phone-free study sessions protect encoding
  • Internal distraction (hunger, fatigue) also causes loss
  • High-yield topics require undivided attention
  • Environment matters — quiet study space
❌ What NOT To Do

Massed Repetition (Cramming)

  • Studying same subject for hours without break
  • Creates massive interference — one topic disrupts another
  • Requires MORE total trials for same outcome
  • Poor long-term retention
  • May work for recognition (MCQ) but fails for recall (essays)
💤 The Sleep Advantage
💤

Sleep is Active Memory Processing Time

Forgetting slows dramatically during sleep because new sensory input (the main cause of interference) is eliminated. The brain actively consolidates STM → LTM during sleep cycles. Never sacrifice sleep the night before exams — it's when your review sessions get "saved".
